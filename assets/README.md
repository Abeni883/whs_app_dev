# Assets für PDF-Export

## Erforderliche Dateien

### Logos:

1. **sbb_logo.png**
   - SBB Logo
   - Empfohlene Größe: 200x80 px
   - Format: PNG mit transparentem Hintergrund
   - Platzierung: Links im Header

2. **achermann_logo.png**
   - Achermann & Co. AG Logo
   - Empfohlene Größe: 200x80 px
   - Format: PNG mit transparentem Hintergrund
   - Platzierung: Rechts im Header

## Platzhalter

Falls die Logos noch nicht vorhanden sind, können Sie diese später hinzufügen.
Das PDF wird auch ohne Logos generiert, zeigt aber Platzhalter an.

## Hinweise

- Verwenden Sie hochauflösende Logos für beste Druckqualität
- Transparente Hintergründe sind empfohlen
- Logos sollten im Format PNG oder SVG vorliegen
