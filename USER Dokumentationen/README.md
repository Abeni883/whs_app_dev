## Wichtiger Hinweis zur Datenbank

Die Anwendung verwendet seit Version 1.0 **SQLite 3** als Datenbank.

Alte Dokumentationen, die **MySQL** erwähnen, sind veraltet und sollten nicht mehr verwendet werden.
